INSTRUCTIONS DÉTAILLÉES:
  1. Installez "SMS Gateway for Android" sur votre téléphone
  2. Ouvrez l'app et activez le serveur HTTP
  3. Assurez-vous que votre téléphone et PC sont sur le même réseau WiFi
  4. Notez l'adresse IP affichée (ex: 192.168.1.100:8080)
  5. Entrez cette URL dans le champ ci-dessus
  6. Cliquez sur "TESTER" pour vérifier la connexion
DÉPANNAGE:
  • Vérifiez que le pare-feu de votre téléphone ne bloque pas
  • Essayez avec http:// au lieu de https://
  • Vérifiez que l'IP est correcte (paramètres → à propos du téléphone → état)
  • Essayez un autre port si 8080 ne fonctionne pas (8081, 9090)
  • Désactivez VPN sur votre téléphone si activé
